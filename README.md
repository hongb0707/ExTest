## 셋업에 관한 절차

- Eclipes
  1. github 사이트에서 프로젝트에 참여하여 이 원격저장소에 push 할수 있는 협업자을 등록
  2. 초대를 받은 팀원이 수락(URL : https://github.com/사용자이름/repository이름/invitations)
  3. 워크스페이스 준비, 서버 세팅, 인코딩 세팅, DB세팅 등 사전 작업
  4. git Repositories 에서 Clone a Git reppository 클릭
  5. URL 에 (https://github.com/hongb0707/ExTest.git) 주소 입력 USER와 Password 에 수락받은 자신 github 아이디, 비밀번호 작성
  6. clone 할 위치, 워크스페이스를 선택
  7. git Repositories 에서 Repository 우클릭후 Pull
